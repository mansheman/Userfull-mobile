--- PASSWORD STORAGE ---
Credentials are stored in SharedPreferences:
  - "EncryptedUsername" -> Base64-encoded username
  - "superSecurePassword" -> AES-encrypted password (hardcoded key)

Both fields can be extracted via:
  adb backup -f backup.ab com.android.insecurebankv2
  # or
  run-as com.android.insecurebankv2 cat shared_prefs/mySharedPreferences.xml

Since the AES key is hardcoded, any attacker can decrypt the stored
password.
