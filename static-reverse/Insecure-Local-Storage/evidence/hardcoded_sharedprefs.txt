This opens the preferences file with WORLD-READABLE permissions on Android < 7 (API 24).
Confirmed: File permissions are rw-rw-rw- (666).

--- Decryption Proof of Concept ---
The hardcoded AES key can be extracted from any decompiled APK:
$ jadx -d output InsecureBankv2.apk
$ grep "super secret key" output/sources/com/android/insecurebankv2/CryptoClass.java
String key = "This is the super secret key 123";

With this key and the all-zeros IV, an attacker can decrypt any password stored
in mySharedPreferences.xml.
